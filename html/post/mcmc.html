Probabilistic Decryption using MCMC Sampling


This project involved developing a probabilistic decryption system to decode encrypted messages efficiently using Markov Chain Monte Carlo (MCMC) techniques. The system leverages a probabilistic language model based on character-level transition probabilities to reconstruct plaintext messages from ciphered text. It demonstrates the application of probabilistic modeling and Bayesian inference in solving cryptography-inspired tasks.

Key Features

  • Probabilistic Language Model: Constructed a bigram frequency matrix to capture statistical relationships between characters.
  • Efficient Sampling Algorithm: Implemented the Metropolis-Hastings algorithm to explore high-dimensional cipher permutations.
  • Bayesian Inference Techniques: Applied log-likelihood scoring to evaluate decoding probabilities.

Tech Stack

  • Programming Language: Python
  • Libraries: NumPy
  • Techniques: Probabilistic Modeling, Markov Chain Monte Carlo (MCMC), Bayesian Inference

Implementation

Bigram Frequency Matrix Construction: Generated a matrix of character-level transition probabilities using War and Peace as the reference text.
MCMC Sampling with Metropolis-Hastings: Designed a sampling algorithm to iteratively explore cipher permutations, with acceptance probabilities calculated using log-likelihood scoring.
Decoding and Evaluation: Decoded encrypted messages by applying the most probable cipher permutation, reducing error rates and enhancing robustness.

Challenges and Solutions

  • Challenge: Ensuring efficient sampling from a high-dimensional probability space.
    Solution: Optimized sampling by tuning parameters such as burn-in period and step interval.
  • Challenge: Capturing accurate transition probabilities for real-world text.
    Solution: Leveraged a large reference text corpus (War and Peace) to train the bigram frequency matrix.

Results

  • Achieved an 80% reduction in decoding error rates compared to baseline approaches.
  • Enhanced decryption efficiency through intelligent cipher permutation exploration.
  • Demonstrated the scalability of MCMC sampling for complex probabilistic inference tasks.
