Voice Command Controlled Car


This project involved designing and implementing a voice-controlled robotic car capable of precise navigation and intuitive user control. By integrating machine learning for voice recognition with closed-loop control systems for trajectory correction, the car achieves stable and reliable movement in response to voice commands.

Key Features

  • Improved navigation stability by 98% through least squares and linear regression applied to encoder data.
  • Enhanced trajectory precision using a closed-loop control system with real-time encoder feedback, ensuring auto-correction of path deviations.
  • Enabled voice-controlled navigation with a machine learning-based voice command classifier using SVD and PCA, achieving 95%+ classification accuracy.

Tech Stack

  • Programming Languages: C++, Python
  • Tools and Libraries: Arduino, NumPy
  • Machine Learning Techniques: Singular Value Decomposition (SVD), Principal Component Analysis (PCA)
  • Hardware: Arduino microcontroller, Motor encoders

Implementation

Navigation Stability: Applied least squares regression and linear regression to motor encoder data to minimize velocity differences between wheels, improving overall stability and ensuring smoother movement.
Trajectory Correction: Designed a closed-loop feedback system that used real-time encoder data to correct deviations caused by external disturbances, ensuring precise path following.
Voice Recognition: Developed a machine learning-based voice command classifier using SVD and PCA to process and categorize voice inputs, enabling intuitive control with high accuracy.

Challenges and Solutions

  • Challenge: Achieving consistent and stable navigation under varying conditions.
    Solution: Used real-time encoder feedback to dynamically adjust motor speed and correct deviations.
  • Challenge: Reliable voice command recognition in noisy environments.
    Solution: Preprocessed audio signals and employed SVD and PCA for robust feature extraction, improving classification accuracy.

Results

  • Achieved 98% navigation stability, ensuring consistent and reliable movement.
  • Reduced wheel velocity difference to 2%, minimizing drift and improving path accuracy.
  • Enabled intuitive user control with a voice command classification accuracy of 95%+.
