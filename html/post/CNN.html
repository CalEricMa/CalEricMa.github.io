Neural Network from Scratch


This project involved building a fully functional neural network framework from scratch using only NumPy. The implementation included forward and backward propagation, multiple activation functions, training logic, and optional support for convolutional layers and batch normalization â€” all designed without using high-level ML libraries like TensorFlow or PyTorch.

Key Features

  • Custom implementation of feedforward neural networks with support for multiple layers and nonlinear activations.
  • Backpropagation algorithm for gradient computation and parameter updates using stochastic gradient descent (SGD).
  • Optional support for batch normalization to stabilize training and improve convergence speed.
  • Numerical gradient checks to validate correctness of backpropagation logic.
  • Extended framework to support convolutional and pooling layers.

Tech Stack

  • Programming Language: Python
  • Libraries: NumPy (no high-level DL libraries used)
  • Concepts: Forward/Backward Propagation, ReLU, Softmax, BatchNorm, SGD

Implementation

The model was trained on the Iris dataset and achieved strong performance despite being built from scratch. Training involved calculating gradients manually via backpropagation, applying loss using cross-entropy, and tuning hyperparameters like learning rate and hidden layer size. Batch normalization was added to reduce internal covariate shift and speed up training. Later iterations added support for convolutional operations and max pooling layers, enabling the framework to scale to image-based tasks.

Challenges and Solutions

  • Challenge: Ensuring gradient correctness without built-in autograd tools.
    Solution: Implemented and passed numerical gradient checks on all layer types.
  • Challenge: Stabilizing training for deeper models.
    Solution: Integrated batch normalization and carefully initialized weights to accelerate convergence.

Results

  • Achieved 96% classification accuracy on the Iris dataset.
  • Improved training speed by 30% after introducing batch normalization and optimized NumPy operations.
  • Successfully scaled framework to include convolutional and max-pooling layers with validated gradient flows.
